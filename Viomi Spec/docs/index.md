

### 项目地址
请访问 [GitHub][1]，你可以 Clone 至本地进行修改并提交。

### 简介
仅供内部同学使用，请勿将相关设计稿外传。
本项目基于 [MKDocs][2] 的 Markdown 工具撰写，MKDocs 需要另外安装 [Material][3] 主题。

图片目前寄存于此 [Viomi UI Image][4] 这个项目之中。

### 文件结构及规范
- Build 后将 `site` 中的文件全部剪切至项目文件夹中，并删除 `site` 的文件夹
- 然后 Commit 并 Push 至主干即可
